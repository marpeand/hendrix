The Open Source API for Climate Data aims to make climate-related data easily accessible to developers. Whether you're building an app for weather forecasting or environmental research, this API provides:

- Real-time temperature, humidity, and wind data.
- Historical climate trends and data analysis tools.
- Geolocation support for targeted queries.

### Features and Tech Stack

1. **Built with Node.js**: High performance and scalability.
2. **MongoDB for Storage**: Efficiently store large datasets.
3. **API Documentation**: Swagger UI for easy integration.

This project empowers developers to build tools that address climate challenges.
